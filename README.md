# Plataforma-CAM-CAD-Data-Frontier

Plataforma de CAM/CAD integrada com Inteligência Artificial e Programação Visual.

## Estrutura do Projeto

- `src/App`: Lógica da Aplicação (Backend / C++ e Python)
- `src/Gui`: Interface Gráfica (Frontend / JS e WebGL)
- `src/shared`: Utilitários compartilhados
- `docs`: Documentação
- `tests`: Testes automatizados
